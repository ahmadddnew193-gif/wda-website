## üé® Design System

### Color Palette
- **Primary Black**: `#000000` - Main background
- **Deep Navy**: `#0a0a2e`, `#1a1a4d` - Gradient backgrounds
- **Neon Blue**: `#3b82f6` - Primary accent
- **Electric Purple**: `#a855f7` - Secondary accent
- **Hot Pink**: `#ec4899` - Tertiary accent
- **Cyan**: `#22d3ee` - Supporting accent

### Typography
- **Headings**: Bold, modern sans-serif (system default)
- **Body**: Clean, readable sans-serif
- **Buttons**: Semibold with tracking
- **Size Scale**: Responsive from mobile to desktop

### Animation Principles
- **Duration**: 0.3s - 2s depending on effect
- **Easing**: `easeInOut`, `easeOut` for natural motion
- **Stagger delays**: 0.1s - 0.2s between elements
- **Infinite loops**: For ambient background effects
- **Scroll-triggered**: Animations fire when in viewport

## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd astro
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

4. Open browser at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
